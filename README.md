# [Rapido Clone]

A clone of the cab booking app, Rapido, built in React Native and Expo SDK 52.

## Tech Stack
- React Native
- React
- Expo
- Zustand + MMKV (persistence)

## Features
- Makes use of the Google Maps SDK for pick and drop locations, show live driver location
- Fare estimatino based on road-distance between two points

## Note
This is a learning project built to explore React Native development.
